# MERN Authentication System

A full-stack **MERN Authentication System** with secure login/signup using **JWT (HttpOnly Cookies)** and **Email OTP verification**.

## Features

- User Registration (Signup)
- User Login & Logout
- JWT Authentication
- Protected Routes (only logged-in users can access)
- Email OTP Verification
- User Profile Data Fetching
- Toast Notifications for errors/success

---

## Tech Stack

**Frontend**
- React.js
- React Router DOM
- Axios
- Tailwind CSS
- React Toastify

**Backend**
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (JsonWebToken)
- Cookie Parser
- Nodemailer (Brevo SMTP)
